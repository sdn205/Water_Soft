# 颜色分类 (Water Sort Puzzle)

一款纯前端的水排序益智游戏，将不同颜色的液体倒入试管，使每个试管只包含同一种颜色。

**在线游玩：[https://water-sort-santonin.pages.dev/](https://water-sort-santonin.pages.dev/)**

## 游戏规则

- 场上有 **10 根试管**（8 根装有混合颜色液体 + 2 根空试管）
- 每根试管最多容纳 **4 层**液体
- 共有 **8 种颜色**：红、橙、黄、绿、青、蓝、紫、粉
- 点击一根试管选中，再点击另一根试管将顶部液体倒入
- 只能将液体倒入空试管或顶部颜色相同的试管
- 当每根非空试管内只有一种颜色时，关卡通过

## 功能特性

- **200 关**逐步解锁，通关前一关后解锁下一关
- **撤销 / 重置**：随时回退操作或重新开始当前关卡
- **最佳纪录**：记录每关最少步数和撤销次数
- **音效**：基于 Web Audio API 的点击、倒水、错误、通关音效（可开关）
- **关卡总览**：分页浏览所有关卡，查看通关状态
- **进度存档**：游戏进度自动保存在 localStorage
- **响应式布局**：适配手机、平板、桌面及横屏模式

## 快速开始

无需构建工具或服务器，直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 方式一：直接双击打开
start index.html

# 方式二：本地服务器
npx serve .
```

## 技术实现

### 关卡生成

- 使用 **种子随机数**（mulberry32 PRNG），关卡编号作为种子，确保同一关卡每次生成相同布局
- 通过哈希函数 `hashSeed(levelNum)` 将关卡编号映射为 PRNG 种子
- 生成时校验：无单色试管、每种颜色至少分布在 2 根试管中、干扰色 ≥ 3

### BFS 求解器

- 使用 **位编码** 将 10 根试管的状态压缩到 5 个 Uint32（80 bit），加速状态比较和存储
- 广度优先搜索寻找最优解，支持：
  - 最少步数求解
  - 多路径枚举
  - 陷阱操作检测（一步走错导致无解）
  - 提示功能（返回最优解的下一步）

### 状态管理

- 游戏进度通过 `localStorage` 持久化，存储键为 `water_sort_progress`
- 撤销功能基于历史快照栈，每步操作前保存完整试管状态
